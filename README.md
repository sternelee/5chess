简单粗暴地实现五子棋小游戏

使用`React`和`Canvas`来实现五子棋小游戏，其中的游戏逻辑参考了[ Web五子棋游戏程序设计](http://www.cnblogs.com/taoweiji/archive/2012/12/09/2810464.html).

## 实现的功能

 - [x] 双人轮流下棋
 - [x] 五子棋胜负逻辑
 - [x] 回退功能
 - [x] 悔棋恢复功能

## `React`方式

只使用`React`编写该游戏，使用`create-react-app`初始化构建，其中打包后的`buld`内的代码已经更改了路径，可以直接打开预览。

## `Canvas`方式

只在参考案例中样式以及回退功能。
